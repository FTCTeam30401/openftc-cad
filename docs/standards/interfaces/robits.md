# Interface: AndyMark RoBits

**System:** imperial · **Fastener:** #10-32 · **Grid:** 0.5 in · Confidence: ✅ (best-specified vendor)

## Pattern

| Feature | Value |
|---------|-------|
| Grid spacing | 0.5 in (12.7 mm) |
| Hole Ø | 0.201 in (5.11 mm) — #10 clearance |
| Gusset resolution | 0.25 in (6.35 mm) |
| Bearing-hole pitch | every 1.5 in (38.1 mm) on 1×1 tube |
| Tubes | 0.5×0.5, 1×0.5, 1×1 in; wall 0.063 in (1.6 mm); 6061-T6 |
| Shaft | 3/8 in hex (9.525 mm), #10-32 both ends |

## Notes
- **AndyMark uses #10-32 hardware broadly** — across RoBits *and* their FRC tubing/gussets. It's the vendor's house thread, not just a RoBits choice.
- RoBits shares the **0.5 in grid + #10-32 thread** with **REV ION** — the two are the most directly interoperable ecosystems (adapters between them are nearly 1:1). See the [hardware-families table](../master-variables.md#1a-cross-vendor-hardware-families).
- VEX also uses a 0.5 in grid but with #8-32 and *square* holes — shares pitch, not thread or hole shape.

Cleanest numbers of any vendor — the target for the first *imperial* hole pattern (`robits_halfinch`).
