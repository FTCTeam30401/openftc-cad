## Notes
- RoBits is intentionally **cross-compatible with VEX's 0.5 in grid** — the main difference is round holes (RoBits) vs. square holes (VEX). This makes RoBits↔VEX adapters mostly a hole-shape problem.
- RoBits' 0.5 in / #10 grid is also close to **REV ION**, not REV DUO.

Cleanest numbers of any vendor — a good target for the first *imperial* hole pattern (`robits_halfinch`).
