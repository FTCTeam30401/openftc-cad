# Structural

Load-bearing frame and connection parts.

**Planned generators (Phase 2):** flat plates, L brackets, U gussets, T gussets, bearing blocks, standoffs, cross members.

Each part declares which mounting standard(s) it supports (REV / goBILDA / RoBits / VEX) using the shared [interface definitions](../../docs/standards/interfaces/). Nothing here invents its own hole spacing.
